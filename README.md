# Каталог Telegram ботов - seraviellex PROJECTS

Сайт-каталог Telegram ботов проекта seraviellex PROJECTS, созданный с использованием Next.js и Tailwind CSS.

## Особенности проекта

- 🎨 Современный дизайн в стиле Telegram
- 📱 Адаптивная вёрстка для всех устройств
- 🚀 Быстрая загрузка благодаря Next.js
- 🎭 Плавные анимации и переходы
- 🔍 Подробная информация о каждом боте
- 📖 Инструкции по использованию в формате HTML

## Технологии

- **Next.js 14** - React-фреймворк для создания веб-приложений
- **React 18** - Библиотека для создания пользовательских интерфейсов
- **Tailwind CSS 3** - Утилитарный CSS-фреймворк
- **PostCSS** - Инструмент для преобразования CSS

## Установка и запуск

### Требования

- Node.js 18+ и npm

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

### Сборка для продакшена

```bash
npm run build
npm start
```

## Добавление новых ботов

Для добавления нового бота в каталог, отредактируйте файл `data/bots.js`:

```javascript
{
  id: 4,
  name: "Название бота",
  username: "имя_бота",
  avatar: "https://ссылка-на-аватар.png",
  welcomeMessage: `Краткое приветственное сообщение...`,
  guideMarkdown: `<h2>Руководство</h2><p>Подробная инструкция...</p>`
}
```
